# BAU AKTS Bilgi Paketi

Bahçeşehir Üniversitesi AKTS Bilgi Paketi ve Ders Kataloğu arayüz projesi.

## Sayfalar
- `index.html`: Ana sayfa
- `arama-sonuclari.html`: Arama sonuçları listeleme sayfası
- `arama-sonuc-yok.html`: Sonuç bulunamadı sayfası
- `programlar.html`: Program listeleme sayfası

## Özellikler
- Responsive tasarım (Mobil uyumlu)
- Bootstrap 5 altyapısı
- Özel CSS stilleri
